Teens for Good — Simple Website

Preview locally

Open `index.html` in a browser. Example (Mac/Linux):

```bash
open index.html
```

Windows: double-click `index.html` or open with your browser.

Deploy

- Push to GitHub and enable GitHub Pages from the repository settings, or
- Host using any static host (Netlify, Vercel, Surge) by connecting the repo.

Notes & Safety

- This site is a starter template. For real contact/donation flows add server-side handling and verified payment processors.
- If members are minors, ensure adult supervision for fundraising and online activities.

Customize

Edit `index.html`, `styles.css`, and `script.js` to add your branding, photos, and event data.
